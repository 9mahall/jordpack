How to Add the "Jord Pack" Modpack to CurseForge Client


Method 1: Importing the Modpack ZIP File

  1. Go to the downloads page and pick up the newest version of the pack.
  2. Open the CurseForge app.
  3. Click the Import Modpack button (usually at the top or under My Modpacks).
  4. Select the ZIP file you downloaded.
  5. Wait for CurseForge to import and set up the modpack. It will appear as a new profile in My Modpacks.
  6. Click Play to launch Minecraft with the imported Jord Pack.
Note: Imported modpacks do not auto-update via CurseForge. For updates, you must repeat the import process with a new ZIP.


Method 2: Adding Mods Manually

  1. Open the CurseForge app and go to the Minecraft section.
  2. Follow the steps in Method 1 to add a new instance. (we will only be using this for its mods folder)
  3. Right Click your current pack and select Open Folder Youll need to do this for both instances current and new.
  4. Drag the jars from the new instance to the mods folder of your current instance.
  5. Skip any that are aleady in the folder and the new ones will transfer. (at this point the new instance you took mods from can be deleted)
  6. Return to CurseForge and click Play on your profile.
Note: Make sure all mods are compatible with your selected Minecraft version and mod loader.(forge & 1.20.1)


Troubleshooting

  • If the modpack does not appear after installation, restart the CurseForge app.
  • If Minecraft crashes, double-check that all mods are compatible and none are missing dependencies.
  • For further help, consult the official CurseForge support or open an issue on our Github.
